# Task Manager App

A simple task manager application built using Node.js and Express and mongo db.

## 🚀 Features
- Add tasks
- Delete tasks
- Update tasks
- Mark tasks as completed

## 🛠️ Tech Stack
- Node.js
- Express.js
- HTML, CSS, JavaScript
- mongo db
